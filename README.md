## :books: Instalação 

Clone o repositório:
```sh
$ git clone https://github.com/dayana-sog/goomer
```

Aceda a pasta do projeto:
```sh
$ cd goomer
```
Instale as dependências:
```sh
yarn
```
Inicie o servidor:
```sh
$ yarn start
```
